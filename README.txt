1. Необходимо скачать и установить docker.
2. Открыть папку с проектом в visual studio code.
3. Настроить виртуальную среду в проекте (.env).
4. Скачать набор необходимых библиотек из файла "requirements.txt".
5. Открыть терминал и прописать в терминале "sudo docker pull postgres", для того чтобы скачать уже готовый образ SQL базы postgres.
6. Прописать в терминале "sudo docker run --name "Название контейнера" -p 5432:5432 -e POSTGRES_USER="Логин" -e POSTGRES_PASSWORD="Пароль" -e POSTGRES_DB="Название базы данных" -d postgres" для создания контейнера с SQL базой.
7. В Visual Studio Code запустить файл "main.py".
8. В браузере перейти по ссылке 127.0.0.1:8080.
